# Telegram-бот по самозанятости (НПД)\n\nИнформационный ассистент для консультаций по вопросам налога на профессиональный доход в РФ.\n\n## Установка\n\n1. Перейдите в папку проекта\n2. Создайте виртуальное окружение: python -m venv venv\n3. Активируйте: venv\\Scripts\\activate (Windows)\n4. Установите зависимости: pip install -r requirements.txt\n5. Создайте .env и заполните ключи\n6. Запустите: python bot.py\n
